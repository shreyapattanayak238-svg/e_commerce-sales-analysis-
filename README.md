# e_commerce-sales-analysis-
project overview-
This project performs exploratory data analysis (EDA) on an e-commerce dataset to understand sales performance, product demand, and customer behavior.
Tools Used-
1. Python
2. Pandas
3. Matplotlib
4. SQL
 Analysis Performed
1. Revenue by Region
2. Revenue by Product Category
3. Quantity Sold per Category
4. Discount Analysis
5. Customer Rating Analysis
Key Insights
1. Sales revenue is evenly distributed across regions.
2. Beauty products generate the highest revenue.
3. Sports category has slightly lower demand.
4. Discounts are applied similarly across categories.
5. Customer satisfaction is consistent across products.
Business Recommendations
1. Increase marketing campaigns for sports products.
2. Offer seasonal discounts and bundle deals for sports items.
3. Promote sports products through fitness influencers and targeted advertising.
